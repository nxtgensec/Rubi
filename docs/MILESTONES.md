# Milestones

## Milestone 1: Architecture Skeleton

- Monorepo structure
- FastAPI app
- Next.js dashboard
- Provider interfaces
- Tool and agent contracts
- Docker Compose

## Milestone 2: Backend Core

- PostgreSQL models and migrations
- Repository pattern
- JWT auth and RBAC
- Audit logs
- Real tool registry
- OpenTelemetry setup

## Milestone 3: AI Orchestration

- Qwen3 adapter
- OpenAI, Anthropic, Gemini, Ollama, and OpenRouter adapters
- Streaming responses
- Tool calling
- Conversation summaries

## Milestone 4: Knowledge And Memory

- PDF, DOCX, FAQ, Markdown, text, and website ingestion
- Chunking and embeddings
- Qdrant indexing
- Caller memory persistence

## Milestone 5: Voice

- Browser voice chat
- Faster Whisper adapter
- Piper and Kokoro TTS adapters
- Interruptions
- VAD and silence detection
- LiveKit/Pipecat integration

## Milestone 6: Telephony

- Provider webhook receiver
- Call answer instructions
- Recording metadata callbacks
- Multilingual call sessions
- SIP provider
- Asterisk integration
- Twilio and Exotel adapters
- Incoming and outbound calls
- Transfer
- Recording

## Milestone 7: Production

- CI/CD
- Docker image publishing
- Secrets management
- Rate limiting
- Observability dashboards
- Horizontal scaling guide
